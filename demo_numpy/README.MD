## 说明

numpy 核心数据结构就是一维或多维的 array。这里把一维的array记为向量，多维的array记为矩阵。

python numpy 数组显示不全：
```
import numpy as np  
np.set_printoptions(threshold=np.inf)  
或者    np.set_printoptions(threshold='nan')
```
